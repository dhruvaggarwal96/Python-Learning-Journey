## 📚 Topics Covered

### What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that organizes code using objects and classes.

### Class

A class is a blueprint or template used to create objects.

```python
class Student:
    pass
```

### Object

An object is an instance of a class.

```python
s1 = Student()
```

### Real-Life Example

* Class → Car
* Objects → BMW, Audi, Tesla

A class defines the structure, while objects are real instances created from that structure.

* OOP helps organize code efficiently.
