# 🚀 Day 4: Lists in Python

Welcome to **Day 4** of my **90-Day Python & Data Science Learning Journey**.

Today I learned one of the most important data structures in Python: **Lists**.

Lists are widely used in Data Analysis and Data Science for storing and managing collections of data such as customer records, sales data, website traffic, student marks, and more.

---

## 📚 Topics Covered

### ✅ List Basics

* Creating Lists
* Accessing Elements
* Positive Indexing
* Negative Indexing
* Modifying Elements

### ✅ List Methods

* `append()`
* `insert()`
* `remove()`
* `pop()`
* `sort()`
* `reverse()`

### ✅ Loops with Lists

* Iterating through lists using `for` loops
* Performing calculations using loops

### ✅ Built-in Functions

* `len()`
* `sum()`
* `max()`
* `min()`

## 🧠 Key Learnings

* Lists are ordered and mutable collections.
* Indexing helps access elements efficiently.
* List methods simplify data manipulation.
* Loops allow processing large collections of data.
* Lists form the foundation for advanced tools like NumPy and Pandas.
