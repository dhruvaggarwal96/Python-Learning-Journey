## 📚 Topics Covered

### What is Inheritance?

Inheritance allows one class to acquire the properties and methods of another class. It promotes code reusability and helps organize programs efficiently.

### Parent Class (Base Class)

A class whose attributes and methods can be inherited by other classes.

### Child Class (Derived Class)

A class that inherits properties and methods from a parent class.

### Method Overriding

A child class can provide its own implementation of a method already defined in the parent class.

### super() Function

Used to access the constructor and methods of the parent class from the child class.
