# Day 14 - Student Performance Dashboard 📊

## Overview

Today I built a **Student Performance Dashboard** using NumPy as part of my 90-Day Data Science Journey.

This project analyzes student marks and generates useful insights such as average marks, topper details, grade distribution, and students performing above average.
