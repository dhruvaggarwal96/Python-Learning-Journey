## 💻 Coding Practice

Created a DataFrame using Pandas:

```python
import pandas as pd

data = {
    "Name": ["Dhruv", "Aman", "Riya"],
    "Age": [20, 21, 19],
    "Marks": [85, 90, 88]
}

df = pd.DataFrame(data)

print(df)
print(df.shape)
print(df.columns)
print(df.info())
```

## 🎤 Key Learnings

* Pandas is used for data manipulation and analysis.
* Series is a one-dimensional data structure.
* DataFrame is a two-dimensional tabular structure.
* `shape` returns rows and columns.
* `columns` displays column names.
* `info()` provides dataset information.
* `describe()` provides statistical summaries.
