---

## 📚 Topics Covered

### 1. Creating Functions

Functions are reusable blocks of code designed to perform specific tasks.

### 2. Function Parameters

Parameters allow data to be passed into a function.

### 3. Return Statements

The `return` keyword sends a value back from a function.

### 4. Local Variables

Variables created inside a function can only be accessed within that function.

### 5. Global Variables

Variables created outside functions can be accessed throughout the program.

### 6. Functions with Conditions and Loops

Functions can be combined with conditions and loops to build powerful programs.
